# FiveM Script Starter Tool

أداة ويب عربية تساعدك تبني هيكل سكربت FiveM بسرعة وبشكل عملي على جهازك.

## المميزات

- توليد `fxmanifest.lua` تلقائي.
- توليد ملفات `client/server/shared` الأساسية.
- اختيار Framework (`QBCore` أو `ESX` أو بدون).
- خيارات إضافية مثل `config.lua` و `locales` وواجهة NUI.
- توليد سطر جاهز لـ `server.cfg`.
- معاينة حيّة لكل ملف داخل الواجهة.
- **تنزيل ZIP جاهز** يحتوي كل الملفات داخل مجلد الريسورس.
- **حفظ مباشر داخل مجلد تختاره** (على Chrome/Edge الداعمين لـ File System Access API).

## التشغيل

```bash
cd fivem-script-builder
python3 -m http.server 4173
```

ثم افتح:

`http://localhost:4173`

## طريقة الاستخدام السريعة

1. عبّي بيانات السكربت.
2. اضغط **توليد الملفات**.
3. اختَر واحدة من الطريقتين:
   - **تنزيل ZIP جاهز** ثم فك الضغط داخل `resources`.
   - **حفظ مباشر داخل مجلد** واختر مجلد `resources` ليتم إنشاء الريسورس تلقائياً.
