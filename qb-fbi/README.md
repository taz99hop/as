# qb-fbi (Police Command Suite)

سكريبت QBCore + qb-target للشرطة بواجهة قيادة مخصصة (نمط شبيه بالصورة المرجعية) مع صلاحيات حسب الرتبة، **ومميزات ميدانية فعلية** بدل مجرد تسجيل بالواجهة.

## المزايا المضمنة (10/10)
1. **التحقيق الجنائي المتقدم**: إنشاء ملف قضية يتضمن تسلسل بصمات / DNA / دم / توقيت وفاة.
2. **AI Dispatch & Backup**: استدعاء وحدتي NPC بدوريات شرطة مع siren ومسار دعم.
3. **التحقيق الصوتي والاعترافات**: حفظ أسئلة/إجابات وتأثيرها على القضية.
4. **Drone & Bridge Cams**: تفعيل كاميرا درون ميدانية + كاميرات جسور قابلة للتبديل من qb-target.
5. **Evidence Tagging**: وسم الأدلة (صورة/فيديو/صوت) وحفظها للنيابة.
6. **K9 System**: كلب شرطة فعلي (spawn) مع أوامر تتبع/شم/بحث.
7. **Smart Pursuit AI**: نشر سبايك ستريب/حاجز مروري بسيارات شرطة/حاجز متحرك.
8. **Command Center UI**: لوحة مباشرة للبلاغات وتوزيع الدوريات.
9. **Police Academy**: حفظ نتائج تدريب القيادة/الرماية/الذكاء.
10. **Shift Report**: تقرير نهاية شفت تلقائي مع مسار تصدير PDF رمزي.

## الرتب والصلاحيات
- Cadet / Officer / Sergeant / Lieutenant / Chief.
- كل وحدة تظهر بالواجهة فقط إذا الرتبة تملك صلاحيتها.

## الأوامر
- `/policehub` فتح واجهة القيادة.
- `/policehub_clear` تنظيف عناصر العمليات (K9/حواجز/كاميرات).

## الاعتمادات
- `qb-core`
- `qb-target`
- `oxmysql`

## ملاحظات
- التخزين محلي في `server/police_data.json`.
- يمكن ربط التصدير PDF فعلياً لاحقاً عبر API خارجي.
